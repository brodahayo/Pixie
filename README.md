## What is Pixie?

Pixie lives in your MacBook's notch and keeps you updated on what Claude Code is doing. It shows:

- **Animated pixel art mascots** — choose from 8 characters (Crab, Robot, Ghost, Cat, Skull, Alien, Sprout, Muddy)
- **Session status** — see when Claude is processing, waiting for input, or needs approval
- **Liquid shimmer progress** — a glassy traveling highlight shows active work
- **Custom notification sounds** — 11 unique synthesized tones plus 14 macOS system sounds
- **Theme colors** — 6 color presets that tint the entire UI

## Install

1. **[Download Pixie-1.0.0.dmg](https://github.com/brodahayo/Pixie/releases/latest/download/Pixie-1.0.0.dmg)**
2. Open the DMG and drag **Pixie** to Applications
3. Launch Pixie — it appears in your notch
4. Grant Accessibility permissions when prompted

## Requirements

- macOS 15.6 or later
- Apple Silicon Mac with a notch display
- Claude Code CLI installed

## Built With

- SwiftUI + AppKit
- Canvas API for pixel art rendering
- Sparkle for auto-updates
